Aangezien het ondertussen een chaos en rommel op mijn PC word met alle code, heb ik ook maar eens een github account aangemaakt zodat ik een beetje versie beheer krijg in mijn prive projectjes en de boel iets meer kan organiseren.

In mijn eigen tijd bouw ik graag aan eigen ideeen en projecten en maak ik me het leven elke keer iets makkelijker. Hiervoor werk ik vaak met verschillende API koppelingen die ik slim aan elkaar knoop waaruit een volledig systeem volgt. Deze plaats ik vervolgens op een eigen domeinnaam en webhosting pakket, welke ik vaak bij Vimexx.nl aanschaf en daar mijn project upload. Dit werkt eigenlijk wel goed, maar als ik een nieuwe fix of update lanceer, houd ik daar geen versie meer van bij.

Vandaar dat ik nu gestart ben met het in GIT plaatsen van mijn code en sites zodat ik met GIT op het hosting pakket(SSH) kan werken en versies kan bijhouden op deze gratis Github dienst. Tot nu toe ideaal en dit scheelt me enorm veel tijd ook keer op keer met het uploaden e.d via FTP wat nu dus via GIT gaat.
